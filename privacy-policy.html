Política de Privacidad

Última actualización: 2025-12-12

1. Responsable del tratamiento

Este servicio es operado por CONFESS.COM.

Contacto: juani.balbiani@gmail.com

2. Datos que recopilamos

  • Número de teléfono de WhatsApp (formato internacional).
  • Mensajes enviados y recibidos a través de WhatsApp dentro del servicio.
  • Datos básicos de perfil: nombre/alias, idioma y zona horaria.
  • Estado de suscripción y datos necesarios para facturación/pagos (según el proveedor de pago utilizado).

3. Para qué usamos los datos

  • Proveer el servicio de conversación por WhatsApp.
  • Gestionar suscripciones, pagos y control de acceso (incluyendo bloqueo por falta de pago).
  • Mejorar la calidad del servicio y la experiencia del usuario.
  • Seguridad, prevención de fraude y cumplimiento de requisitos legales o de la plataforma.

4. Proveedores y terceros

Para operar el servicio podemos utilizar proveedores externos, por ejemplo:

  • Meta / WhatsApp Cloud API (mensajería).
  • Proveedor de IA (por ejemplo, OpenAI) para generar respuestas.
  • Procesadores de pago (por ejemplo, Stripe/MercadoPago).
  • Infraestructura de hosting y base de datos.

No vendemos tus datos personales.

5. Conservación de datos

Conservamos los datos el tiempo necesario para operar el servicio y cumplir obligaciones legales. El usuario puede solicitar la eliminación de sus datos en cualquier momento (ver sección 7).

6. Seguridad

Aplicamos medidas razonables de seguridad para proteger los datos. Sin embargo, ningún sistema es 100% infalible.

7. Derechos y eliminación de datos

El usuario puede solicitar acceso, rectificación o eliminación de sus datos. Para solicitar eliminación, ver instrucciones en Eliminación de datos.

8. Cambios a esta política

Podemos actualizar esta política. La fecha de “Última actualización” indicará los cambios.
